# Keyboard Shortcuts Reference

All keyboard shortcuts can be customized in [Settings](70-settings.md).

## File Operations

| Shortcut | Action |
|----------|--------|
| **Ctrl+O** | Open file |
| **Ctrl+S** | Save file |
| **Ctrl+Shift+S** | Save as |
| **Ctrl+Q** | Quit application |

## Machine Control

| Shortcut | Action |
|----------|--------|
| **Ctrl+H** | Home all axes |
| **Ctrl+R** | Soft reset |
| **Space** | Pause / Resume stream |
| **Escape** | Stop stream (E-Stop) |

## Jog Controls

| Shortcut | Action |
|----------|--------|
| **W** | Jog Y+ (forward) |
| **S** | Jog Y- (backward) |
| **A** | Jog X- (left) |
| **D** | Jog X+ (right) |
| **Q** | Jog Z+ (up) |
| **Z** | Jog Z- (down) |

> Jog keys move the machine by the currently selected step size. See [Machine Control](10-machine-control.md) for step size options.

## Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+Z** | Undo |
| **Ctrl+Y** | Redo |
| **Ctrl+C** | Copy |
| **Ctrl+V** | Paste |
| **Ctrl+X** | Cut |
| **Ctrl+A** | Select all |
| **Ctrl+F** | Find |
| **Delete** | Delete selected |
| **Home** | Go to line start |
| **End** | Go to line end |
| **Ctrl+Home** | Go to file start |
| **Ctrl+End** | Go to file end |

## Designer Shortcuts

| Shortcut | Action |
|----------|--------|
| **Ctrl+G** | Group selected shapes |
| **Ctrl+Shift+G** | Ungroup |
| **Ctrl+C** | Copy shapes |
| **Ctrl+V** | Paste shapes |
| **Delete** | Delete selected shapes |
| **Escape** | Deselect all |

## Visualizer

| Shortcut | Action |
|----------|--------|
| **+** / **-** | Zoom in / out |
| **0** | Fit to view |
| **Arrow keys** | Pan view |

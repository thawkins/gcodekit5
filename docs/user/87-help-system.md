# Help System

## Overview

GCodeKit5 includes a built-in help system with documentation for all features, accessible directly from the application.

## Accessing Help

- **Help menu** — Open the help browser from the application menu
- **Help buttons** — Many panels and CAM tools have a Help button that opens context-sensitive help for that feature
- **Keyboard** — Use the configured help shortcut (see [Settings](70-settings.md))

## Help Browser

The help browser displays markdown-formatted documentation topics:

- **Topic list** — Browse all available help topics
- **Navigation** — Click links within topics to jump to related documentation
- **Cross-references** — Topics link to related topics using `help:topic_id` links

## Available Topics

The help system covers:

| Category | Topics |
|----------|--------|
| **Connection** | Serial, TCP/IP, and WebSocket setup |
| **Machine Control** | Jogging, homing, coordinate systems |
| **File Operations** | Opening, saving, and importing files |
| **G-Code Editor** | Editing, search, and execution tracking |
| **Visualizer** | 3D preview, camera controls, display options |
| **CAM Tools** | One topic per tool (box maker, puzzle, drill press, bitmap engraver, vector engraver, spoilboard surfacing, spoilboard grid, Gerber converter, speeds & feeds) |
| **Designer** | Shapes, import, toolpath generation |
| **Macros** | Creating and running macros |
| **Settings** | Application configuration |
| **Shortcuts** | Keyboard shortcut reference by category |
